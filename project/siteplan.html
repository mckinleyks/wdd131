Bucket List Travel Site Plan

Site Name

Bucket List Travels

This name represents hopes and dreams. Memories and experiences made and had with loved ones.

Optional domain availability: BucketListTravels.com

Site Purpose

The site provides travel guides, destination highlights, and planning tools to plan your next memory.

Scenarios

  • What are the top travel destinations for families?
  • How can I plan a budget-friendly international trip?

Color Schema

Primary Color: Blue (rgb(10, 100, 164)) - Used for headings and accents

Secondary Color: Earthy Green (rgb(43, 128, 43)) - Used for buttons and highlights

Typography

Heading Font: Merriweather (used for all headings)

Body Font: Nunito Sans (used for body text and paragraphs)
